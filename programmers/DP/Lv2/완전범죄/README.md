# ❌ programmers : 완전범죄 (DP , Lv.2)

A 도둑과 B 도둑이 여러 물건을 나눠 훔친다.

각 물건마다:

* A가 훔치면 A 흔적 증가
* B가 훔치면 B 흔적 증가

가 발생한다.

[목표]

모든 물건을 반드시 훔치되:
A 흔적 < n
B 흔적 < m

을 만족해야 한다.

A 흔적의 최솟값 을 구하는 문제.

[핵심 특징]

각 물건마다 선택:

1. A가 훔침
2. B가 훔침

즉 매 물건당 2가지 선택 존재.

[DP 핵심 아이디어]

dp[b] 의 의미:

B 흔적이 b일 때,
가능한 최소 A 흔적

[점화식]

현재 물건:

(aTrace, bTrace)

일 때:

[A가 훔침]

A += aTrace
B 변화 없음

[B가 훔침]

A 변화 없음
B += bTrace

dp[B] = A
