# 📌 알고리즘 레포 Commit 규칙

기본 형식

```
(이모지) 플랫폼: 문제번호 문제이름 (알고리즘, 난이도)
```

예

```
✅ boj: 2178 미로탐색 (BFS, Silver1)
❌ boj: 20920 영단어 암기는 괴로워 (HashMap, Silver3)
🔁 boj: 1463 1로 만들기 (DP, Silver3)
```
