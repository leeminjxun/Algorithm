# ❌ BOJ: 2231 분해합 (완전탐색, Bronze2)

https://www.acmicpc.net/problem/2231

# 풀이 과정

216 이 N 일 경우, N 의 생성자는 198, 207 이다.

우리는 가장 작은 생성자를 찾아야하므로, 1부터 시작하여 N - 1 까지 생성자의 조건에 부합하는지 순차적으로 확인해야한다.

하지만 1부터 비교할 필요는 없다.

위의 예시에서 216 이 N 일 경우에,

세 자리수에 대해서 최댓값은 9, 9, 9 = 27 일 것이다.

만일 216 의 생성자 A 라는 수가 있을 때,

분해합의 수가 최대인 27 과 더했을 경우 216보다 작게 나올 수 없을 것이다.

그러므로 A + 27 >= 216 이여야 하고, A >= 216 - 27 이다.
