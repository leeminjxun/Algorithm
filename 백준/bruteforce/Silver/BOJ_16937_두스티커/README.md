# ❌ BOJ: 16937 두 스티커 (완전탐색, Silver3)

https://www.acmicpc.net/problem/16937

# 풀이 과정

크기가 H X W 인 격자 판에 임의의 스티커 2개를 붙히고 최댓값을 구하는 문제.
(조건 : 스티커는 90도 회전이 가능하다.)

스티커 N 개에 대해서 row(H) 와 col(W) 값이 주어진다.

각 스티커 2개를 선택하고 4번의 비교를 진행해야한다.

1. (r1, c1), (r2, c2) 순으로 스티커를 붙히는 경우 (원본)
2. (c1, r1), (r2, c2) 순으로 스티커를 붙히는 경우 (첫번째 스티커를 90도 회전)
3. (r1, c1), (c2, r2) 순으로 스티커를 붙히는 경우 (두번째 스티커를 90도 회전)
4. (c1, r1), (c2, r2) 순으로 스티커를 붙히는 경우 (모든 스티커 90도 회전)

두 스티커를 가로로 놓는 경우 -> (r1 + r2) 는 H 보다 작거나 같아야하고, (c1, c2) 중 최댓값이 W 보다 작거나 같아야한다.
두 스티커를 세로로 놓는 경우 -> (c1 + c2) 는 W 보다 작거나 같아야하고, (r1, r2) 중 최댓값이 H 보다 작거나 같아야한다.
