# ❌ BOJ: 14501 퇴사 (완전탐색 / dfs, Silver3)

https://www.acmicpc.net/problem/14501

# 풀이 과정

오늘부터 N+1일째 되는 날 퇴사를 하기 위해서, 남은 N일 동안 최대한 많은 상담을 하려고 한다.

if(day >= N + 1) 최종 합이 max 인지 계산한다.

완전탐색에서 선택지는 2가지 이다.
1. 오늘 상담을 받는다.
   - 상담을 받기 위해서는, 오늘 상담을 받고 끝나는 날짜가 N + 1 보다 작거나 같아야한다.
2. 오늘 상담을 받지 않는다.

위의 선택지를 가지고 재귀를 이어나간다.

static void dfs() {
    if(day + T[day] <= N + 1)   dfs(상담 받기);
    dfs(상담 안받기)
}
