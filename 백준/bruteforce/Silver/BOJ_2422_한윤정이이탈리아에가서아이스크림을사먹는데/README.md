# ❌ BOJ: 2422 한윤정이 이탈리아에 가서 아이스크림을 사먹는데 (완전탐색, Sliver4)

https://www.acmicpc.net/problem/2422

# 풀이 과정

N 종류의 아이스크림이 있고, 아이스크림에는 1 ~ N 까지 번호가 매겨져있다.

아이스크림을 섞어 먹는데, 어떤 조합으로는 섞어 먹지 말아야한다.

처음 풀이 시에, 3중 for 문으로 숫자 3개를 만들어
String 클래스의 cotains 메서드를 사용하였지만 메모리 초과로 실패하였다.

해당 문제에서는 boolean 2차원 배열을 생성하여 N + 1 만큼 할당한다.

그 후, 섞어 먹으면 안되는 두 조합을 배열의 index 로 설정하고 true 를 할당한다.

똑같이 3중 for 문을 돌려 i, k, j 세 숫자 변수를 배열 인덱스에 대입하여 true 가 나오는지 체크한다.
