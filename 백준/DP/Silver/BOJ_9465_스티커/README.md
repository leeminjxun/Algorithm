# ✅ BOJ: 9465 스티커 (DP, Silver1)

https://www.acmicpc.net/problem/9465

# 풀이 과정

2 x N 크기의 스티커들이 있을 경우, 하나의 스티커를 떼면 그 스티커 주변의 상하좌우 에 위치한 스티커들이 찢어진다.

각 스티커에는 점수가 매겨져 있고, 스티커의 점수가 최대가 되도록 스티커는 사용하는 경우를 구한다.

dp 를 사용해서 풀이할 경우 각 행에 대하여 경우의 수를 생각해보아야한다.

1. 각 행의 첫번째, 두번째 줄의 스티커를 떼지 않는 경우 -> dp[0][i]
2. 각 행의 첫번째 줄 스티커를 뗀 경우 -> dp[1][i]
3. 각 행의 두번째 줄 스티커를 뗀 경우 -> dp[2][i]

1번의 경우, 이전 상태는 다음과 같다.
1. 이전 행에서 첫번째 줄 스티커를 뗀 경우
2. 이전 행에서 두번째 줄 스티커를 뗀 경우

최댓값을 구해야하기 때문에 위 두 경우 중 최댓값을 가지는 경우를 dp[0][i] 에 더한다.

2번의 경우, 이전 상태는 다음과 같다.
1. 두번째 줄 스티커를 뗸 경우
2. 스티커를 떼지 않은 경우

마찬가지로 두 경우 중 최댓값을 구하고, 현재 행 스티커 점수를 합해준다.

3번의 경우, 위와 동일하다.
