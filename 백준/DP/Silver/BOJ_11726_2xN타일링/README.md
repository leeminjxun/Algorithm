# ✅ BOJ: 11726 2xN 타일링 (DP, Silver3)

https://www.acmicpc.net/problem/11726

# 풀이 과정

2 x N 크기의 직사각형을 1 x 2 와 2 x 1 으로 채우는 방법의 수를 구하라.

처음 블럭을 채워 넣을 때 1 x 2 와 2 x 1 를 채워 넣는 경우의 수로 나눈다.

## 1 x 2 일 때 -> dp[i][1]
직사각형을 모두 채워야하므로 1 x 2 블럭을 2개 넣어야한다.
그러므로 2 x 2 블럭을 채워넣는다.

남은 직사각형의 빈칸 : 2 x (n - 2)

다음 직사각형의 빈칸은 1 x 2 2 개를 넣는 경우의 수와 2 x 1 1 개를 넣는 경우의 수로 나뉜다.

## 2 x 1 일 떄 -> dp[i][0]
2 x 1 블럭 1개만 채워 넣는다.

남은 직사각형의 빈칸 2 x (n - 1)

다음 직사각형의 빈칸은 1 x 2 2 개를 넣는 경우의 수와 2 x 1 1 개를 넣는 경우의 수로 나뉜다.

# 예시 풀이

만일 N 이 4 일 경우 배열 인덱스 4번째에서 다음과 같은 경우의 수가 존재한다.

이전 배열에서 2 x 1 블록이 있을 경우 -> 현재 index - 1
이전 배열에서 2 x 2 블록이 있을 경우 -> 현재 index - 2

그렇다면 index 4 이전에 2개의 블록이 있는 경우와 1개의 블록이 있는 경우 모두 더해줘야한다 -> dp[0][4] + dp[1][4]

dp[0][4] = dp[0][4 - 1] + dp[1][4 - 1]
dp[1][4] = dp[0][4 - 2] + dp[1][4 - 2]
