https://www.acmicpc.net/problem/1912

# 풀이 과정

예제 )

3
26 40 83
49 60 57
13 89 99

1번 집에서 26 을 선택했을 경우 -> 2번 집은 60, 57 중 더 작은 값을 선택한다.
1번 집에서 40 을 선택했을 경우 -> 2번 집은 49, 57 중 더 작은 값을 선택한다.
1번 집에서 83 을 선택했을 경우 -> 2번 집은 49, 60 중 더 작은 값을 선택한다.

점화식

dp[i + 1][0] = Math.min(dp[i][1], dp[i][2]);
dp[i + 1][1] = Math.min(dp[i][0], dp[i][2]);
dp[i + 1][2] = Math.min(dp[i][0], dp[i][1]);
