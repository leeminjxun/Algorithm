# ✅ BOJ: 1309 동물원 (DP , Silver1)

https://www.acmicpc.net/problem/1309

# 풀이 과정

N = 4 일 경우

1   1 
2   2 -> 1 + 1
5   5 -> 1 + 2 + (1 * 2)
12  12 -> 1 + 5 + (2 * 2 + 1 * 2)


1 + (12 + 5 + 2 + 1) * 2 = 41

사자 우리를 반으로 쪼개 N 개의 배열으로 볼 수 있다. 어차피 양 쪽 경우의 수는 대칭이다.

세로 3번째 우리에 사자가 들어갈 경우, 경우의 수는 다음과 같다.

3번째 우리에만 사자를 넣는 경우, 2번째 우리 대각선에서 오는 경우, 1번째 우리 양쪽에서 오는 경우

그러므로 dp[3] += dp[2] + 2 * dp[1] 이 된다.
dp[4] += dp[3] + (dp[2] + dp[1]) * 2 가 된다

점화식은 다음과 같다.

dp[N + 1] += dp[N] + (acc * 2) [acc] 는 N - 1 이전 경우의 수들의 합이다.
acc += dp[N]
