# 📌 BOJ: 11722 가장 긴 감소하는 부분 수열 (DP, Silver2)

# https://www.acmicpc.net/problem/11722

# 풀이 과정

수열 A 주어졌을 경우, 감소하는 부분 수열 중에 가장 긴 값을 구해야한다.

해당 문제는 수열 배열 dp 배열 그리고 반복문을 이용하여 최댓값을 비교한다.

# 예시 풀이

수열 A = {40, 50, 30, 20, 10} 일 경우 가장 긴 감소하는 부분 수열은 40, 30, 20, 10 이 된다.

반복문을 돌리며 각 배열 값을 seq 에 저장하고, dp 배열에는 해당 인덱스 이전 값들을 비교하며 연속되는 감소 부분 수열이 있는지 확인다.

처음 단계를 건너뛰고 수열 요소인 30에 대해 확인해보자.

value = 30, index = 3
seq = {40, 50}
dp ={0, 0}

seq 인덱스를 내림차순으로 조회해본다.

비교 index = 2 의 경우
seq[2] = 50 이므로 이는 현재 index 의 값 30보다 크다. 그러므로 dp[3] = Math.max(dp[3], dp[2] + 1) 진행
