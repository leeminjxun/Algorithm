# 🔁 BOJ: 2293 동전1 (DP, Silver1)

# https://www.acmicpc.net/problem/2293

## 이전 동전의 경수의 수를 저장하여 연결할 수 있다.

## ex) 1원과 2원으로 5원을 만드는 경우의 수

## 1원의 경우
## 1원 2원 3원 4원 5원
## 1  1   1   1  1

## 2원의 경우
## 1원 2원 3원 4원 5원
## 1   2  2   3  3

## 위와 같은 결과값 도출 이유
## 1원 -> 경우의 수 1원 1개
## 2원 -> 경우의 수 1원 2개, 2원 1개
## 3원 -> 경우의 수 1원 3개, 2원 1개와 1원 1개
## 4원 -> 경우의 수 1원 4개, 2원 2개, 1원 2개와 2원 2개
## -> 4원의 경우 2원 메모리 활용
## 1원 2개 + 2원 한개 / 2원 1개 + 2원 1개
## 해서 기존 경우의 수 1원 4개 + 2원의 경우의 수(1원 2개, 2원 1개)
