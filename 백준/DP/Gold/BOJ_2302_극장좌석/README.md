[ 좌석번호 개수 2 ]

{12} 21 
- {12} 는 좌석번호 개수 1 일 때의 수에 2번 좌석을 붙힌 것이다.
- 21 는 좌석번호 개수 0 일 때의 수에 2번 좌석으로 시작하는 경우의 수이다.

[ 좌석번호 개수 3 ]

{123 213} 132
- {123 213} 는 좌석번호 개수 2 일 때의 수에 3번 좌석을 붙힌 것이다.
- 132 는 좌석번호 개수 1 일 때의 수에 3번 좌석을 이어붙힌 경우의 수이다.

따라서 위의 경우의 수를 조합해보았을 때, 
해당 문제에서는 피보나치 수열의 원리를 따르고 있다.

dp[i] = d[i - 1] + dp[i - 2] (i > 1)

이 때 VIP 좌석은 고정 좌석이므로, 각각의 VIP 전 좌석의 개수인 경우의 수만 고려하여 곱해주면 된다.

res *= dp[currentVip - lastVip - 1]
