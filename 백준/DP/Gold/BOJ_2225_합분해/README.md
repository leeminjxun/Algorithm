# ✅ BOJ: 2225 합분해 (DP , Gold5)

https://www.acmicpc.net/problem/2225

# 풀이 과정

0부터 N까지의 정수 K개를 더해서 그 합이 N이 되는 경우의 수를 구하는 프로그램을 작성하시오.

덧셈의 순서가 바뀐 경우는 다른 경우로 센다(1+2와 2+1은 서로 다른 경우). 또한 한 개의 수를 여러 번 쓸 수도 있다.

N = 3, K = 3

N = 1
0   0
1   1
2   2
3   3

N = 2
0   0 0 
1   1 0, 0 1
2   1 1, 2 0, 0 2
3   1 2, 2 1, 3 0, 0 3

dp[K + 1][N + 1] 

경우의 수는 다음과 같이 나눌 수 있다.

1. 정수 1개를 골랐을 때 0 ~ N 이 되는 경우의 수
2. 정수 2개를 골랐을 때 0 ~ N 이 되는 경우의 수 -> 1의 경우를 이용한다.

if 정수 1개를 골랐을 때 0과 1이 되는 경우의 수는 0과 1를 골른 경우밖에 없다.

이후 정수 2개를 골랐을 때 1이 되는 경우의 수는,
정수 1개를 골랐을 때
0이 되는 경우의 수에서 다음 수를 1 을 고르는 경우의 수
1이 되는 경우의 수에서 다음 수를 0 을 고르는 경우의 수

위 두가지가 된다.

점화식으로 나타내면 다음과 같다.

dp[2][1] = dp[1][0] + dp[1][1]
-> dp[1 + 1][1 + 0 or 0 + 1];
