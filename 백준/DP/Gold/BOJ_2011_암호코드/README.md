# ❌ BOJ: 2011 암호코드 (DP , Gold5)

https://www.acmicpc.net/problem/2011

# 풀이 과정

해당 문제에서 고려해야할 사항은 다음과 같다.

1. 선택 숫자의 자릿수 (1의 자리인지 10의 자리인지)
2. 해석 가능한 범위 (1의 자릿수면 1 ~ 9 까지, 10의 자릿수면 10~26)
3. 0의 처리 (100, 001, 010 과 같은 숫자들은 해석이 불가하다.)

dp[i] 는 i번째 자리까지 해석했을 때의 경우의 수 이다.

분기 조건은 다음과 같다.

- 1의 자릿수 선택 
  - 현재 숫자가 1~9 라면, 이전까지의 경우의 수를 더한다. 
  - dp[i] += dp[i - 1]
- 10의 자릿수 선택
  - 현재 숫자와 이전 숫자를 합친 숫자가 10~26 이라면, i - 2 까지의 경우의 수를 더한다.
  - dp[i] += dp[i - 2]
