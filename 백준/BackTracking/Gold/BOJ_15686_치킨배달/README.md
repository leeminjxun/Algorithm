# 🔁 BOJ: 15686 치킨 배달 (백트레킹 / 조합, Gold5)

https://www.acmicpc.net/problem/15686

# 풀이 과정

`5 2
0 2 0 1 0
1 0 1 0 0
0 0 0 0 0
2 0 0 1 1
2 2 0 1 2`

위와 같이 입력 이 주어졌을 경우,

각 1 의 위치에 해당하는 집과 가장 가까운 치킨 집을 계산하여 도시의 치킨 거리(각 집에 대해서 M 개의 치킨 거리의 합) 을 출력해야한다.

입력 값의 배열에서 집과 치킨집 의 위치를 따로 저장한다.

`static List<int[]> houses = new ArrayList<>();
static List<int[]> chickens = new ArrayList<>();`

그 이 후에 조합을 구현하기 위한 백트레킹을 이용하여 M 개의 치킨집을 고르는 경우의 수에 대해 계산을 진행한다.

위의 예시의 백트레킹 과정은 다음과 같다.

```
초기: [?, ?]

[0, ?]
[0, 1] → 계산
[0, 2] → 계산
[0, 3] → 계산
[0, 4] → 계산

[1, ?]
[1, 2] → 계산
[1, 3] → 계산
[1, 4] → 계산

[2, ?]
[2, 3] → 계산
[2, 4] → 계산

[3, ?]
[3, 4] → 계산
```
