# ❌ BOJ: 1931 회의실 배정 (Greedy, Gold5)

https://www.acmicpc.net/problem/1931

# 풀이 과정

현재 회의 의 종료 시각과 다음 회의의 시작 시각을 비교해야한다.

1. 정렬 -> 빨리 끝나는 회의 순서로 정렬한다. 끝나는 시간이 같다면 시작 시작이 빠른 순으로 정렬한다.

2. endTime = 0 으로 초기화 하고, 정렬된 회의의 시작시간과 비교한다.

3. 회의 시작 시간이 endTime 보다 같거나 크면 
   - endTime 을 해당 회의 종료 시간으로 설정
   - 회의 추가
