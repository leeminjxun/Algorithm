# ❌ BOJ: 1202 보석 도둑 (Greedy, Gold2)

https://www.acmicpc.net/problem/1202

# 풀이 과정

1. 상덕이가 털 보석점에는 보석이 총 N개 있다.
2. 각 보석은 무게 Mi와 가격 Vi를 가지고 있다. 상덕이는 가방을 K개 가지고 있고, 각 가방에 담을 수 있는 최대 무게는 Ci이다. 
3. 가방에는 최대 한 개의 보석만 넣을 수 있다.

가격이 높으면서 무게가 가장 낮은 순으로 정렬한다.
가방은 가장 작은 무게가 들어갈 수 있는 가방 순으로 정렬한다.

무엇이 효율적인가? -> 작은 무게의 높은 가격의 보석을 담아야한다. 하지만 가방의 용량 또한 고려해야한다.

[해결 방안]

1. 보석들의 무게를 기준으로 오름차순 정렬한다.
가방 또한 오름 차순 정렬을 진행한다.
2. 가방과 보석들의 무게를 비교한다.
   - 가방의 무게보다 보석의 무게가 작거나 같은 경우, pq 에 보석의 가격을 넣는다.
3. 이 후, pq 의 가장 앞단을 totalPrice 에 더한다. (pq 는 우선순위가 값이 높은 순으로 정렬되도록 정의한다)
