# ❌ BOJ: 1339 단어 수학 (Greedy, Gold4)

https://www.acmicpc.net/problem/1339

# 풀이 과정

GCF + ACDEB를 계산한다고 할 때,

A = 9, C = 8, G = 7, D = 6, E = 5, B = 4, F = 3

해당 문제를 풀이하면서, 간단하게 가장 긴 단어를 배열으로 정렬하여 맨 앞에 오는 단어들을 기준으로 숫자를 부여하면 될 것이라 생각했다.

하지만 위와 같이 풀이 했을 경우에,

ABBBB, BBBB, BBB, BB, B 와 같은 케이스에는 적용이 불가했다.

해당 문제를 풀이할 때 가중치라는 개념을 사용하여 풀이해야했다.

단어 길이에 비례하여, 길이가 4 라면 10^4 인 1,000 의 가중치를 가지는 알파벳으로 정의해야한다.

우리는 전체 알파벳의 가중치를 계산하여, 가중치가 큰 순서대로 숫자만 부여하면 된다.

다시, GCF + ACDEB를 계산한다고 할 때,

A 의 가중치 10,000 * 9 = 90,000
C 의 가중치  1,010 * 8 = 8,080
D 의 가중치    100 * 7 = 700
G 의 가중치    100 * 6 = 600
E 의 가중치     10 * 5 = 50
B 의 가중치      1 * 4 = 4
F 의 가중치      1 * 3 = 3

총합 = 99,437
