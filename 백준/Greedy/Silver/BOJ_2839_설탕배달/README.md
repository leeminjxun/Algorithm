# 🔁 BOJ: 2839 설탕 배달 (Greedy, Silver4)

https://www.acmicpc.net/problem/2839

# 풀이 과정

N 을 5 와 3 으로 만들 수 있는 최소 개수 를 구하라.

18 -> 5 5 5 3, 즉 4 가 최소가 된다.

우리는 3을 최소로 쓰는 경우의 수를 구해야한다.

그렇다면 5로 나누는 경우를 먼저 가져간 후 5 로 전부 나눠지지 않는다면, 3 을 빼준다.

18 % 5 은 0 이 아니다. -> 18 - 3 = 15, count = 1
15 % 5 는 0 이다 -> count += 15 / 5, count = 4
