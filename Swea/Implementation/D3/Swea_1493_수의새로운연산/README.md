# 풀이과정

## 1. 숫자 배치 규칙의 이해
```
(1,1) = 1

(1,2) = 2   (2,1) = 3

(1,3) = 4   (2,2) = 5   (3,1) = 6

(1,4) = 7   (2,3) = 8   (3,2) = 9   (4,1) = 10
...
```

같은 대각선에 위치 해 있는 점들의 합은 항상 같다.

* 1 : (1,1) → x+y=2
* 2,3 : (1,2), (2,1) → x+y=3
* 4,5,6 : (1,3), (2,2), (3,1) → x+y=4

## 2. 좌표 를 숫자로 변환하는 방식
좌표 (x, y) 가 주어졌을 때

대각선 번호 d = x + y - 1 이다.

* 1 : (1,1) → x+y=2 -> 2 - 1 = 1
* 2,3 : (1,2), (2,1) → x+y=3 -> 3 - 1 = 2
* 4,5,6 : (1,3), (2,2), (3,1) → x+y=4 4 - 1 = 3

## 3. 대각선의 시작 숫자 와 끝 숫자

각 대각선의 시작 숫자는 다음과 같다.
1 2 4 7 16 ...

점화식 -> d * (d - 1) / 2 + 1

각 대각선의 끝 숫자는 다음과 같다.
1 3 6 10 15 ...

점화식 -> d * (d + 1) / 2 + 1

## 4. 숫자 -> 좌표

n -> (x, y)

여기서 n 이 포함되는 대각선 을 찾아야한다.

while(대각선 끝 점 < n) 대각선 끝 점이 n 보다 커질 경우 n 은 해당 대각선에 포함된다.

대각선 을 찾았다면 해당 대각선의 시작 점을 찾아야한다.

대각선 시작 점 구하는 공식은 "3. 대각선의 시작 숫자 와 끝 숫자" 에서 증명하였다.

이후 n - 대각선 시작점 을 하여 n 이 대각선 시작점으로 부터 얼마나 떨어져있는지를 구한다.

x = 1 + offset
y = d - offset

## 5. 좌표 -> 숫자

좌표를 받았으니 대각선 번호를 구할 수 있다.

d = x + y - 1

n = d * (d - 1) / 2 + x

대각선 시작점에 x 를 더해주는 이유는 시작점에 대해서 x 만큼 떨어질수록 x 만큼 오르기 때문이다.
