# ❌ SWEA: 3495 승자 예측하기 (구현 / 패턴 , D4)

# 풀이과정

[ Alice 가 이기기 위한 방법 ]

Bob 이 2x 를 골라도 N 을 넘게해야한다.

2x > N  이여야만 하고 B 가 고르는 x 는 N/2 + 1 이여야한다.

[ Bob 이 이기기 위한 방법 ]

Alice 가 N/2 + 1 을 불렀을 때 내가 N 을 고를 수 밖에 없는 상황이 나오지 않도록,
((N/2 + 1) / 2) - 1 을 불러야한다.

[ 예시 ]

N = 100 일 때,

1. 내가 이기기 위해서는 다음 사람이 100을 초과하는 숫자를 부르도록 해야한다.

현재 턴인 사람이 51 을 부르면 다음 사람은 102 를 불러 지게 되기 때문이다.

2. 반대로 현재 턴의 상대는 내가 51 을 부르지 못하도록 24 라는 숫자를 부른다.

그렇게 되면 현재 턴인 사람은 최대 49 라는 숫자만 부르게 된다.

위의 두 가지 경우를 반복 했을 경우,

51 24 13 5 3 .. 이 나오게 된다.
