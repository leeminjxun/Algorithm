# ❌ SWEA: 4408 자기 방으로 돌아가기 (구현 , D4)

# 풀이과정

room 배열을 생성 후, 각 인덱스 마다 지나가는 값들을 계산하고 최대값을 가지는 인덱스의 값을 구한다.

[유의사항]

방 을 지날 때 마주보는 방향 모두를 지난다.

1 3 5 7 9
2 4 6 8 10

5 번 방에서 9 번 방으로 돌아갈 때 5 6 7 8 9 까지가 아닌 마지막에 10 까지 지나치도록 구성해야한다.

start == 짝수 인 경우 start - 1
end == 홀수 인 경우 end + 1

for(i = start; i <= end; i++) {
    room[i]++;
}
