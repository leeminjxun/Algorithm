# ❌ SWEA: 2806 N-Queen (BackTracking , D3)

# 풀이과정

[조건]
1. Queen 은 같은 행, 열 대각선에 겹치면 안된다.
2. N X N 의 체스판에 N 개의 Queen 을 배치해야한다.

위의 조건을 가진 문제에서 말하는 힌트는 다음과 같다.

각 행 or 열에 1개의 QUEEN 이 배치되어야만 한다.

해당 문제에서는 위의 힌트를 가지고 DFS 백트레킹을 이용한다.

[분기 조건]

행을 기준으로 하는 배열 visited[N] 을 생성하고, 각 행에서 N 까지의 열 을 선택하는 경우로 분기를 나눈다.

[분기 실행 조건]

1. 현재 row 의 이전 값들(0 ~ row - 1) 까지 현재 row 에서 선택된 col 과 같은 값이 있는지 확인
2. 현재 row 와 선택된 col 값과 겹치는 대각선 값이 있는지 확인

대각선 검증 식은 다음과 같다.

r1 - r2 == c1 - c2 일 경우 각 좌표 값은 같은 대각선에 존재한다.
