# ❌ SWEA: 1244 최대상금 (BackTracking , D3)

# 풀이과정

완전탐색 + DFS 로 풀 수 있는 문제.

DFS 실행 시 depth 마다 숫자 조합을 저장할 수 있도록 HashSet 배열을 생성한다.

숫자 교체는 우선 숫자를 String 으로 받고, toCharArray 를 사용하여 배열 형태로 다시 저장한다.

완전탐색 -> 2중 for 문을 통해 i가 1 일 때, j : 2 ~ arr.length()

num = "123"
K = 2

의 경우

depth 0 -> {123}

[123] depth 1 -> {213, 321, 132}

[213] depth 2 -> {123, 312, 231} -> 여기까지만 봐도 됨, 밑에는 중복
[321] depth 2 -> {213, 123, 312}
[132] depth 2 -> {312, 231, 123}
