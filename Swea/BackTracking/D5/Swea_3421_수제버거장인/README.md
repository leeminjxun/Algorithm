# ❌ SWEA: 3421 수제버거 장인 (BackTracking , D5)

# 풀이과정

[핵심]

궁합이 맞지 않는 두 재료 i, j 를 boolean 2차원 배열의 인덱스로 칭한다.

boolean[][] badComb -> badComb[i][j] = true;

[백트래킹 분기]

분기 조건은 depth 에 대하여 현재 재료를 선택하느냐, 하지 않느냐 이다.

현재 재료를 선택했을 때는 visited 배열에 재료에 대한 인덱스를 true 상태로 바꾼다.

[조합 확인]
현재 재료를 선택할 수 있는 조건은 궁합이 맞지 않는 재료가 현재 조합에 있는지 확인하는 것이다.

이는 현재 상태의 visited 배열을 확인하여야한다.

1 ~ N 까지 반복문을 돌리며 visited[i] 가 true 이고, badComb[i][idx] (idx 는 현재 선택하려는 재료이다.) 가 true 라면,

해당 재료는 현재 조합에서 선택할 수 없다.
