# 풀이과정

[가지치기]

주어진 P(i, j) 를 100 으로 나눠 소수로 만든다.

이 때 DFS 내부에서 P 를 곱하면 곱할수록 percent 는 작아진다.

한 분기에서 percent <= res 라면, 해당 분기에서 남은 직원들의 확률 곱해도 커질 수 없다.

예)

현재 percent = 0.4
남은 확률 = 0.9, 0.8
최종 확률 = 0.4 * 0.9 * 0.8 = 0.288

즉, 확률을 곱하면 현재의 확률보다 같거나 작을 뿐이다.
